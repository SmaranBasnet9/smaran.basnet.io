AdRoll Opens Beta for an MCP Server

June 15, 2026 Summarized by Smaran

AdRoll has opened a beta for an MCP (Model Context Protocol) server that lets tools like ChatGPT and Claude connect directly to AdRoll account data.

In practice, that means a marketer could ask an AI assistant something like "create a retargeting campaign draft for recent site visitors" and have it pull real account data to build that draft, instead of switching into the AdRoll dashboard, exporting numbers, and building the campaign by hand.

It's a beta, so the natural-language layer is a draft-first tool right now — it proposes a campaign, a human still has to review and launch it. But the direction is the interesting part: AI assistants are starting to become a genuine interface layer for ad platforms, not just a chat sidebar bolted onto an existing dashboard.

Worth watching for any team running AdRoll: this is the kind of beta access that's worth requesting early, since the value is mostly in how it fits your specific account structure and naming conventions — not something a press release can fully convey.

Original announcement: AdRoll
